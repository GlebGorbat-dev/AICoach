# Инструкция по установке и запуску фронтенда

## Быстрый старт

1. **Установите зависимости:**
```bash
cd frontend
npm install
```

2. **Настройте переменные окружения:**
   
   Создайте файл `.env.local` в папке `frontend`:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:8000
   ```

3. **Запустите бэкенд:**
   
   Убедитесь, что ваш FastAPI бэкенд запущен на `http://localhost:8000`

4. **Запустите фронтенд:**
```bash
npm run dev
```

5. **Откройте браузер:**
   
   Перейдите на [http://localhost:3000](http://localhost:3000)

## Структура приложения

### Страницы:
- `/` - Главная (редирект на /login или /chats)
- `/login` - Страница входа
- `/register` - Страница регистрации
- `/chats` - Список всех чатов
- `/chats/[chatId]` - Интерфейс конкретного чата

### Компоненты:
- `Button` - Кнопка с неоновым эффектом
- `Input` - Поле ввода с темной темой
- `Message` - Компонент сообщения (пользователь/AI)
- `ChatCard` - Карточка чата в списке

### API клиент:
Все эндпоинты бэкенда покрыты в `lib/api.ts`:
- Аутентификация (register, login)
- Чаты (getAll, get, create, delete, updateTitle)
- Агент (sendMessage, getMessages, getDocumentsCount)

## Особенности дизайна

- **Темная тема** - черный фон (#0a0a0a) с темными поверхностями
- **Неоновые красные акценты** - #ff006e с эффектом свечения
- **Минималистичный дизайн** - чистый и современный интерфейс
- **Анимации** - плавные переходы и эффекты свечения

## Решение проблем

### Ошибка подключения к API
Убедитесь, что:
1. Бэкенд запущен на `http://localhost:8000`
2. В `.env.local` указан правильный URL
3. CORS настроен в бэкенде (уже настроен в вашем коде)

### Ошибки TypeScript
Убедитесь, что установлены все зависимости:
```bash
npm install
```

### Проблемы со стилями
Проверьте, что Tailwind CSS правильно настроен:
```bash
npm run build
```
